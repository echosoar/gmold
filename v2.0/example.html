Variable var

Conditional Statement -> if-else-endif

<{if:ifistrue}> Variable ifistrue is True
<{else}> Variable ifistrue is False
<{endif}>

Conditional Statement -> if-elseif-else-endif

<{if:ifvalueA}> Variable ifvalueA is True
<{elseif:ifvalueB}> Variable ifvalueB is True
<{else}> Variable ifvalueA 和 ifvalueB 均is False
<{endif}>

Conditional Statement -> switch-case-endswitch

<{switch:switchval}> Variable switchval neither A nor B
<{case:'A'}> Variable switchval is A
<{case:'B'}> Variable switchval is B
<{endswitch}>

Conditional Statement -> for[start-end]-endfor

<{for[0<4]}> <{$i}> <{forvar.$i}>
<{endfor}>
