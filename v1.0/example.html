Conditional Statement -> if-elseif-else-endif

<{if:ifvalueA}> Variable ifvalueA is True
<{elseif:ifvalueB}> Variable ifvalueB is True
<{else}> Variable ifvalueA 和 ifvalueB 均is False
<{endif}>

Conditional Statement -> switch-case-endswitch

<{switch:switchval}> Variable switchval neither A nor B
<{case:'A'}> Variable switchval not A
<{case:'B'}> Variable switchval not B
<{endswitch}>
